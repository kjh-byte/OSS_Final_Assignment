# ⚔️ SW 모험가 길드: 전직 퀘스트 안내소

정보융합학부 학생들을 위한 맞춤형 전공 및 기술 스택 추천 웹 애플리케이션입니다. 

## 💡 프로젝트 기획 배경
흔한 추천 앱에서 벗어나, 정보융합학부의 전공 로드맵을 'RPG 모험가의 전직 퀘스트'라는 세계관으로 재해석했습니다. 각 기술 스택(C/C++, Java 등)의 특징을 게임 아이템에 비유하여 학습자가 재미있게 자신의 전공 진로를 탐색할 수 있도록 설계했습니다.

## 🛠 Tech Stack
* **Frontend**: Streamlit
* **Backend**: FastAPI
* **Containerization**: Docker, Docker Compose
* **Deployment**: AWS EC2

## 🚀 프로젝트 실행 방법
본 프로젝트는 Docker 환경에서 구동됩니다. 최상위 디렉토리에서 다음 명령어를 실행하세요.
